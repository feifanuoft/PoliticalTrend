There are 8 .py files.

A2_Start.py is given in the assigment folder and I used its functions.

Read_file can read the .tsv file directly.

CNN_model.py is a baseline model class

args_parameter.py contains the parameter needed during training, if you want to change the parameter, please change the value in this .py file.

The training function is in the CNN_train.py

If you want to train the model: please run the CNN_main.py diectly.

If you want to explore the meaning of the parameters in the linear neuron, please run the Closet_word.py

If you want to get the best parameter to train the model, please run the cnn_best_parameter.py 

datasets are in the "data" folder
